# 📘 React Manual — Aprendizaje Incremental con VitePress

> Manual diseñado para aprender React + TypeScript de forma progresiva, práctica y profesional.

---

## 🎯 Objetivo

Este repositorio contiene un manual estructurado para:

- Aprender React desde cero hasta nivel profesional
- Construir conocimiento de forma **incremental**
- Aplicar conceptos mediante **proyectos prácticos**
- Adoptar buenas prácticas reales de desarrollo

---

## 🧠 Enfoque pedagógico

Este manual sigue principios clave:

- ✅ Aprendizaje progresivo (sin saltos de complejidad)
- ✅ Cada tema usa solo conceptos ya aprendidos
- ✅ Refuerzo continuo mediante ejercicios
- ✅ Enfoque práctico orientado a proyectos reales

## 🚀 Instalación

Clonar el repositorio:

```bash
git clone git@personal:blonder413/b-React-19.git
cd b-React-19
```

> En mi caso estoy usando un archivo de configuración para poder usar ssh en mi cuenta personal y otra llave para proyectos laborales.

Instalar dependencias:

```bash
pnpm install
```

---

## ▶️ Desarrollo

Ejecutar entorno local:

```bash
pnpm docs:dev
```

---

## 🏗️ Build

Generar versión estática:

```bash
pnpm docs:build
```

Previsualizar:

```bash
pnpm docs:preview
```
