# 📘 Manual de React — Aprendizaje Incremental

> Aprende React + TypeScript desde cero hasta un nivel profesional, paso a paso.

---

## 🚀 Bienvenido

Este manual está diseñado para que aprendas React de una forma diferente:

- Sin saltos de complejidad  
- Sin conceptos mágicos  
- Sin copiar código sin entender  

Aquí **cada tema depende del anterior**, para que construyas conocimiento sólido y duradero.

---

## 🧠 Cómo usar este manual

Para obtener el máximo provecho:

1. Sigue el orden de las secciones  
2. No saltes temas  
3. Completa todos los ejercicios  
4. Repite conceptos clave en proyectos pequeños  

---

## 🧱 Estructura de aprendizaje

El contenido está dividido en secciones progresivas:

- Fundamentos
- React básico
- State y eventos
- Render dinámico
- Hooks
- Routing
- Consumo de APIs
- Arquitectura
- Testing
- Tooling
- Proyecto final

---

## ⏱️ Enfoque incremental

Cada sección:

- Introduce **solo lo necesario**
- Usa **solo conocimientos previos**
- Refuerza con ejercicios prácticos

---

## 🧪 Cómo practicar

En cada sección encontrarás:

- ✅ Ejercicios básicos  
- ✅ Ejercicios integrados  
- ✅ Mini proyectos  

Ejemplo de flujo:

> Aprendes useState → construyes formulario → lo mejoras con validaciones → lo integras en una vista


---

## 🎯 Objetivo final

Al terminar este manual podrás:

- Construir aplicaciones reales con React
- Entender cómo organizar proyectos
- Aplicar buenas prácticas profesionales
- Escribir código mantenible y testeable
